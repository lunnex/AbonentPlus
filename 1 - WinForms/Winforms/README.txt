Эта программа реализует CRUD - операции над 3 таблицами базы данных.
Программа содежит в себе 3 grid view - по одному для каждой таблицы.
Операции удаления вставки и редактирования реализованны через контекстное меню. Необходимо выбрать нужную запись в таблице и кликнуть правой клавишей для вызова контекстного меню.
Для таблицы requests предусмотрены фильтры. Для их вызова так же необходимо вызвать контекстное меню.
